[猜數字範例程式](https://repl.it/@shes50103/game)

由 0~9 選三個數字隨機排列 <br>
要猜到有哪些 `數字` 以什麼 `順序` 排列出來 <br>
 
數字對 +1B <br>
數字位置都對 +1A <br>

## 範例
當答案是 135 的時候 <br>

012 => 0A1B <br>
102 => 1A0B <br>
315 => 1A2B <br>

最後猜出 3A 時結束遊戲
